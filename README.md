## AMAZON TRADER PRO

amazon ürünlerini takip eder ve fiyat değişikliklerinde sizi uyarır. .py dosyası .exe olarak derlenmiştir ve çalıştırmak için bir .bat dosyası kullanılır.

İÇERİK

- amazon_trader.py – ana python dosyası
- run-amazon-trader.bat – .exe çalıştırmak için gerekli bat dosyası
- data.json – takip edilen ürün verilerini saklar (program otomatik oluşturur)
- amazon_trader.exe - .py dosyasının programlaştırılmış arayüzlü .exe hali 

KURULUM VE KULLANIM

1. projeyi indirin veya klonlayın
2. tüm dosyaları aynı klasörde tutun
3. run-amazon-trader.bat dosyasını çalıştırın
   - bu dosya .exe programının doğru şekilde çalışmasını sağlar
4. program açıldığında:
   - ürün linklerini ekleyin
   - hedef fiyat belirleyin
   - Start butonuna basarak takip etmeye başlayın
5. fiyat değişikliklerinde popup mesaj ve ses ile uyarı alırsınız

ÖNEMLİ NOTLAR

- program arka planda çalışacak şekilde tasarlanmıştır
- python dosyasını manuel çalıştırmak yerine .exe ve .bat dosyalarını kullanın
- takip edilen veriler data.json dosyasında saklanır
